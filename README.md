# Resignation Website

This project is a static website designed to help users create and display resignation letters. It includes templates, styles, and scripts to facilitate the process of writing and formatting resignation letters.

## Project Structure

```
resignation-website
├── src
│   ├── index.html          # Main homepage of the resignation website
│   ├── resignation.html    # Displays the content of the resignation letter
│   ├── templates
│   │   └── letter-template.html  # Template for creating resignation letters
│   ├── css
│   │   ├── styles.css      # Main styles for the website
│   │   └── print.css       # Styles for printing the resignation letter
│   ├── js
│   │   └── main.js         # JavaScript for interactivity
│   └── data
│       └── example.json    # Example data for demonstration
├── package.json            # npm configuration file
├── .gitignore              # Files and directories to ignore in version control
└── README.md               # Documentation for the project
```

## Getting Started

To get started with the resignation website, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   cd resignation-website
   ```

2. **Install dependencies**:
   If you have npm installed, run:
   ```
   npm install
   ```

3. **Open the website**:
   Open `src/index.html` in your web browser to view the homepage.

## Features

- **Resignation Letter Template**: Use the letter template to create personalized resignation letters.
- **Print Styles**: The website includes styles specifically for printing resignation letters.
- **Interactivity**: JavaScript functionality for handling user input and dynamic content updates.
